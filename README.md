# AF-TT-RDQN
This repository contains the data and code for the paper: "Action-factorized Rainbow deep Q-network with token Transformer for computation offloading in edge computing-enabled Internet of Ships".
Data: The parameter data is provided in the following .xlsx files: Three feature groups.xlsx, Simulation parameters.xlsx, Network parameters.xlsx.
Code: The various components of the algorithm, the simulation environment, and the core implementation logic are located in the accompanying Python (.py) files.
